# Frontend

Next.js App Router Projekt mit Tailwind CSS für das Unternehmens-Frontend.

## Entwicklung

```bash
npm install
npm run dev
```

Die Anwendung startet standardmäßig unter `http://localhost:3000`.

## Struktur

- `app/` – Seiten und Layouts
- `components/` – wiederverwendbare UI-Bausteine
- `app/contact` / `app/impressum` – statische Informationsseiten
