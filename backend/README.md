# Backend

Ein schlanker Express-Server, der Beispiel-Endpunkte für das Frontend bereitstellt.

## Entwicklung

```bash
npm install
npm run dev
```

Die API ist anschließend unter `http://localhost:4000` erreichbar.

## Endpunkte

- `GET /api/health` – Gesundheitscheck
- `GET /api/products` – Beispielproduktdaten
