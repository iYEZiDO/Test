# NextLevel Plattform

Dieses Repository enthält eine Beispiel-Struktur für ein Unternehmensprojekt mit getrennten Bereichen für Frontend, Backend und Dokumentation.

## Struktur

- `frontend/` – Next.js App Router Projekt mit Tailwind CSS und responsivem Basislayout (Header, Footer, Produkt-Grid, Kontakt- und Impressum-Seiten).
- `backend/` – Express.js Server mit Beispiel-API-Endpunkten.
- `docs/` – Statische Legacy-Seiten und Assets.

## Erste Schritte

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

Die Dokumentation befindet sich im Verzeichnis `docs/`.
